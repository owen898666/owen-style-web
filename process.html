拍攝流程

Four Simple Steps · 簡單四步，輕鬆完成你的拍攝

Process

如何開始

從第一次聯絡到收到作品，只需要四個步驟

01

Step One

初步諮詢

透過 Facebook、Instagram 或 LINE 聯絡我，告訴我你想要拍什麼、什麼時候、需要多久。

我會回覆你我的可用檔期，並回答你所有的問題。 這個階段完全免費，不需要任何承諾。

這個階段你會獲得

  • ✓ 我的可用檔期
  • ✓ 完整的服務方案與報價
  • ✓ 拍攝風格參考
  • ✓ 任何問題的解答

02

Step Two

確認方案 & 簽約

選定方案之後，我會寄出拍攝合約。合約內容包含拍攝時間、地點、 交付項目、修改範圍等，清楚透明，沒有隱藏費用。

確認合約內容後，支付訂金（通常是總價的 50%）， 檔期就正式確定了。

你需要提供的

  • ✓ 拍攝日期與地點
  • ✓ 拍攝內容（婚禮流程、想要的風格等）
  • ✓ 訂金 NT$ 5,000
  • ✓ 簽署拍攝合約

03

Step Three

拍攝當天

我會提前抵達，確認光線與環境。你只需要放輕鬆、做自己， 我會在旁邊捕捉每一個自然的表情與瞬間。

婚禮拍攝我會從新娘準備開始，一路拍到宴會結束。 中間我會適時引導，但大部分時間你不需要理會我—— 只要享受當下。

我的承諾

  • ✓ 提前到場勘景
  • ✓ 安靜、不干擾的拍攝風格
  • ✓ 自然光優先
  • ✓ 捕捉每一個真實情感瞬間

04

Step Four

交付作品

婚禮結束後 4-6 週內，你會收到一封包含精修檔案的下載連結。 所有原始檔我也會整理好放在雲端，方便你隨時下載保存。

交付後如果有小幅調整需求（色溫、裁切）， 都可以提出，我會全力配合。

交付內容

  • ✓ 精修檔案（JPG，高解析度）
  • ✓ 所有原始 RAW 檔
  • ✓ 線上相簿連結
  • ✓ 交付後小幅調整服務
